Πολιτική Απορρήτου

Τελευταία ενημέρωση: 28 Μαΐου 2024

Αυτή η πολιτική απορρήτου περιγράφει τις πολιτικές και τις διαδικασίες μας για τη συλλογή, τη χρήση και τη διαφάνεια των πληροφοριών σας κατά τη χρήση της υπηρεσίας και σας ενημερώνει για τα δικαιώματα απορρήτου σας και τον τρόπο που ο νόμος σας προστατεύει.

Χρησιμοποιούμε τα προσωπικά σας δεδομένα για να παρέχουμε και να βελτιώνουμε την υπηρεσία. Χρησιμοποιώντας την υπηρεσία, συμφωνείτε με τη συλλογή και χρήση των πληροφοριών σύμφωνα με αυτή την πολιτική απορρήτου.

Ερμηνεία και Ορισμοί

Ερμηνεία

Οι λέξεις με αρχική κεφαλαία γράμματα έχουν ορισμένα νοήματα σύμφωνα με τις ακόλουθες συνθήκες. Οι ακόλουθοι ορισμοί θα έχουν τον ίδιο νόημα ανεξαρτήτως εάν εμφανιστούν στον ενικό ή στον πληθυντικό αριθμό.

Ορισμοί

Για τους σκοπούς αυτής της πολιτικής απορρήτου:

  • Λογαριασμός σημαίνει ένα μοναδικό λογαριασμό που δημιουργήθηκε για εσάς για να έχετε πρόσβαση στην υπηρεσία μας ή σε μέρη της υπηρεσίας μας.

  • Συνδεδεμένη Εταιρεία σημαίνει μια εταιρεία που ελέγχει, ελέγχεται από, ή είναι υπό κοινό έλεγχο με μια εταιρεία, όπου "έλεγχος" σημαίνει κυριότητα 50% ή περισσότερων των μετοχών, του μετοχικού συμφέροντος ή άλλων αξιογράφων με δικαίωμα ψήφου για την εκλογή διοικητικών ή άλλων διαχειριστικών αρχών.

  • Εταιρεία (αναφέρεται ως "η εταιρεία", "εμείς", "εμείς", ή "μας" σε αυτή τη συμφωνία) αναφέρεται σε αυτό τον ιστότοπο.

  • Μπισκότα είναι μικρά αρχεία που τοποθετούνται στον υπολογιστή, την κινητή συσκευή σας ή οποιαδήποτε άλλη συσκευή από έναν ιστότοπο, περιέχοντας λεπτομέρειες της ιστορίας περιήγησής σας σε αυτόν τον ιστότοπο ανάμεσα στις πολλές χρήσεις του.

  • Συσκευή σημαίνει οποιαδήποτε συσκευή που μπορεί να έχει πρόσβαση στην υπηρεσία, όπως ένας υπολογιστής, ένα κινητό τηλέφωνο ή μια ψηφιακή ταμπλέτα.

  • Προσωπικά Δεδομένα είναι οποιαδήποτε πληροφορία που σχετίζεται με ένα αναγνωρίσιμο ή αναγνωρίσιμο ατομικό.

  • Υπηρεσία αναφέρεται στον ιστότοπο.

  • Πάροχος Υπηρεσίας σημαίνει κάθε φυσικό ή νομικό πρόσωπο που επεξεργάζεται δεδομένα εκ μέρους της εταιρείας. Αναφέρεται σε εταιρείες ή άτομα τρίτων που απασχολούνται από την εταιρεία για να διευκολύνουν την υπηρεσία, να παρέχουν την υπηρεσία εκ μέρους της εταιρείας, να εκτελούν υπηρεσίες σχετικές με την υπηρεσία ή να βοηθούν την εταιρεία να αναλύει τον τρόπο με τον οποίο χρησιμοποιείται η υπηρεσία.

  • Δεδομένα Χρήσης αναφέρεται σε δεδομένα που συλλέγονται αυτόματα, είτε δημιουργούνται από τη χρήση της υπηρεσίας είτε από την υποδομή της υπηρεσίας ίδια (για παράδειγμα, η διάρκεια μιας επίσκεψης σε μια σελίδα).

  • Εσείς σημαίνει το άτομο που έχει πρόσβαση ή χρησιμοποιεί την υπηρεσία, ή την εταιρεία ή άλλο νομικό πρόσωπο εκ μέρους του οποίου το άτομο αυτό έχει πρόσβαση ή χρησιμοποιεί την υπηρεσία, ανάλογα.

Συλλογή και Χρήση των Προσωπικών σας Δεδομένων

Τύποι Συλλεγμένων Δεδομένων

Προσωπικά Δεδομένα

Κατά τη χρήση της υπηρεσίας μας, μπορεί να σας ζητήσουμε να μας παρέχετε ορισμένες προσωπικά αναγνωρίσιμες πληροφορίες που μπορούν να χρησιμοποιηθούν για να επικοινωνήσουμε ή να σας αναγνωρίσουμε. Οι προσωπικά αναγνωρίσιμες πληροφορίες μπορεί να περιλαμβάνουν, αλλά δεν περιορίζονται σε:

  • Δεδομένα Χρήσης

Δεδομένα Χρήσης

Τα δεδομένα χρήσης συλλέγονται αυτόματα κατά τη χρήση της υπηρεσίας.

Ότανπροσπελάσετε την υπηρεσία μέσω κινητής συσκευής, μπορούμε αυτόματα να συλλέξουμε ορισμένες πληροφορίες, συμπεριλαμβανομένων, αλλά όχι περιοριστικά, του τύπου της κινητής συσκευής που χρησιμοποιείτε, του μοναδικού αναγνωριστικού της κινητής συσκευής σας, της IP διεύθυνσης της κινητής συσκευής σας, του λειτουργικού συστήματος της κινητής συσκευής σας, του τύπου του φυλομετωπικού περιηγητή ιστού που χρησιμοποιείτε, μοναδικά αναγνωριστικά συσκευών και άλλα διαγνωστικά δεδομένα.

Επίσης, μπορεί να συλλέξουμε πληροφορίες που στέλνει το πρόγραμμα περιήγησής σας όταν επισκέπτεστε την υπηρεσία ή όταν έχετε πρόσβαση στην υπηρεσία μέσω μιας κινητής συσκευής.

Τεχνολογίες Παρακολούθησης και Μπισκότα

Χρησιμοποιούμε μπισκότα και παρόμοιες τεχνολογίες παρακολούθησης για να παρακολουθούμε τη δραστηριότητα στην υπηρεσία μας και να αποθηκεύουμε συγκεκριμένες πληροφορίες. Οι τεχνολογίες παρακολούθησης που χρησιμοποιούνται μπορεί να περιλαμβάνουν δείκτες, ετικέτες και σενάρια για να συλλέγουν και να παρακολουθούν πληροφορίες και να βελτιώνουν και να αναλύουν την υπηρεσία μας. Οι τεχνολογίες που χρησιμοποιούμε μπορεί να περιλαμβάνουν:

  • Μπισκότα. Ένα μπισκότο είναι ένα μικρό αρχείο που τοποθετείται στη συσκευή σας. Μπορείτε να διατάξετε τον περιηγητή σας να αρνείται όλα τα μπισκότα ή να υποδεικνύει όταν ένα μπισκότο αποστέλλεται. Εάν δεν αποδεχτείτε τα μπισκότα, ενδέχεται να μην μπορέσετε να χρησιμοποιήσετε κάποια μέρη της υπηρεσίας μας. Εάν δεν προσαρμόσετε τις ρυθμίσεις του περιηγητή σας για να αρνείστε τα μπισκότα, η υπηρεσία μας μπορεί να χρησιμοποιεί μπισκότα.
  • Δείκτες Διαδικτύου. Ορισμένες ενότητες της υπηρεσίας μας και τα email μας μπορεί να περιέχουν μικρά ηλεκτρονικά αρχεία γνωστά ως δείκτες διαδικτύου (επίσης αναφέρονται ως διαφανείς gifs, ετικέτες pixel και μονόχρωμα gifs) που επιτρέπουν στην εταιρεία να, για παράδειγμα, μετρήσει τους χρήστες που έχουν επισκεφθεί αυτές τις σελίδες ή έχουν ανοίξει ένα email και για άλλα σχετικά στατιστικά του ιστότοπου (για παράδειγμα, καταγραφή της δημοτικότητας μιας συγκεκριμένης ενότητας και επαλήθευσης της ακεραιότητας του συστήματος και του διακομιστή).

Τα μπισκότα μπορεί να είναι "Μόνιμα" ή "Συνεδρίας". Τα μόνιμα μπισκότα παραμένουν στον προσωπικό σας υπολογιστή ή συσκευή όταν είστε εκτός σύνδεσης, ενώ τα μπισκότα συνεδρίας διαγράφονται αμέσως μόλις κλείσετε τον περιηγητή σας στον ιστό. Μπορείτε να μάθετε περισσότερα για τα μπισκότα στο άρθρο του ιστότοπου TermsFeed.

Χρησιμοποιούμε τόσο συνεδρίας όσο και μόνιμα μπισκότα για τους παρακάτω σκοπούς:

  • Αναγκαία / Ενσωματωμένα Μπισκότα

    Τύπος: Μπισκότα συνεδρίας

    Διαχειρίζεται από: Εμάς

    Σκοπός: Αυτά τα μπισκότα είναι απαραίτητα για να σας παρέχουν τις υπηρεσίες που είναι διαθέσιμες μέσω του ιστότοπου και για να σας επιτρέψουν να χρησιμοποιήσετε μερικές από τις λειτουργίες του. Βοηθούν στην αυθεντικοποίηση των χρηστών και στην αποτροπή της απάτης της χρήσης των λογαριασμών χρηστών. Χωρίς αυτά τα μπισκότα, οι υπηρεσίες που έχετε ζητήσει δεν μπορούν να παρασχεθούν, και χρησιμοποιούμε αυτά τα μπισκότα μόνο για να σας παρέχουμε αυτές τις υπηρεσίες.

  • Πολιτική Μπισκότων / Ειδοποίηση Συγκατάθεσης για Μπισκότα

    Τύπος: Μόνιμα μπισκότα

    Διαχειρίζεται από: Εμάς

    Σκοπός: Αυτά τα μπισκότα προσδιορίζουν εάν οι χρήστες έχουν αποδεχτεί τη χρήση μπισκότων στον ιστότοπο.

  • Μπισκότα Λειτουργικότητας

    Τύπος: Μόνιμα μπισκότα

    Διαχειρίζεται από: Εμάς

    Σκοπός: Αυτά τα μπισκότα μας επιτρέπουν να θυμόμαστε τις επιλογές που κάνετε κατά τη χρήση του ιστότοπου, όπως την αποθήκευση των στοιχείων σύνδεσής σας ή τις προτιμήσεις γλώσσας σας. Ο σκοπός αυτών των μπισκότων είναι να σας παρέχουν μια πιο εξατομικευμένη εμπειρία και να αποφύγετε να ξανακαταχωρήσετε τις προτιμήσεις σας κάθε φορά που χρησιμοποιείτε τον ιστότοπο.

    Για περισσότερες πληροφορίες σχετικά με τα μπισκότα που χρησιμοποιούμε και τις επιλογές σας σχετικά με τα μπισκότα, επισκεφθείτε την Πολιτική Μπισκότων μας ή την ενότητα μπισκότων της Πολιτικής Απορρήτου μας.

    Χρήση των Προσωπικών Δεδομένων Σας

    Η εταιρεία μπορεί να χρησιμοποιεί προσωπικά δεδομένα για τους παρακάτω σκοπούς:

    • Για να παρέχει και να διατηρεί την υπηρεσία μας, συμπεριλαμβανομένου του παρακολούθησης της χρήσης της υπηρεσίας μας.

    • Για να διαχειριστεί τον λογαριασμό σας: για να διαχειριστείτε την εγγραφή σας ως χρήστης της υπηρεσίας. Τα προσωπικά δεδομένα που παρέχετε μπορεί να σας δώσουν πρόσβαση σε διαφορετικές λειτουργίες της υπηρεσίας που είναι διαθέσιμες για εσάς ως εγγεγραμμένος χρήστης.

    • Για την εκτέλεση μιας σύμβασης: την ανάπτυξη, τη συμμόρφωση και την εκτέλεση της σύμβασης αγοράς για τα προϊόντα, τα αντικείμενα ή τις υπηρεσίες που έχετε αγοράσει ή οποιαδήποτε άλλη σύμβαση μαζί μας μέσω της υπηρεσίας.

    • Για να επικοινωνήσει μαζί σας: για να επικοινωνήσει μαζί σας μέσω email, τηλεφώνου, SMS ή άλλων ισοδύναμων μορφών ηλεκτρονικής επικοινωνίας, όπως ειδοποιήσεις push από μια εφαρμογή κινητού, σχετικά με ενημερώσεις ή ενημερωτικές επικοινωνίες που αφορούν τις λειτουργίες, τα προϊόντα ή τις συμβεβλημένες υπηρεσίες, συμπεριλαμβανομένων ενημερώσεων ασφαλείαικών, όταν είναι απαραίτητο ή λογικό για την υλοποίησή τους.

    • Για να σας παρέχει νέα, ειδικές προσφορές και γενικές πληροφορίες σχετικά με άλλα αγαθά, υπηρεσίες και εκδηλώσεις που προσφέρουμε και είναι παρόμοια με αυτά που έχετε ήδη αγοράσει ή ρωτήσει για αυτά, εκτός εάν έχετε επιλέξει να μην λαμβάνετε τέτοιες πληροφορίες.

    • Για να διαχειριστεί τις αιτήσεις σας: για να παρακολουθεί και να διαχειρίζεται τις αιτήσεις σας προς εμάς.

    • Για επιχειρηματικές μεταβιβάσεις: Μπορούμε να χρησιμοποιήσουμε τις πληροφορίες σας για να αξιολογήσουμε ή να πραγματοποιήσουμε μια συγχώνευση, απόσχιση, αναδιάρθρωση, επανοργάνωση, διάλυση ή άλλη πώληση ή μεταφορά ορισμένων ή όλων των περιουσιακών μας στοιχείων, είτε ως ενεργή επιχείρηση είτε ως μέρος πτώχευσης, αποσαφήνισης ή παρόμοιας διαδικασίας, όπου τα προσωπικά δεδομένα που κατέχουμε για τους χρήστες της υπηρεσίας μας βρίσκονται ανάμεσα στα μεταφερόμενα περιουσιακά στοιχεία.

    • Για άλλους σκοπούς: Μπορούμε να χρησιμοποιήσουμε τις πληροφορίες σας για άλλους σκοπούς, όπως ανάλυση δεδομένων, αναγνώριση των τάσεων χρήσης, καθορισμός της αποτελεσματικότητας των προωθητικών μας εκστρατειών και για την αξιολόγηση και βελτίωση της υπηρεσίας, των προϊόντων, των υπηρεσιών, του μάρκετινγκ και της εμπειρίας σας.

    Μπορούμε να μοιραστούμε τις προσωπικές σας πληροφορίες στις παρακάτω καταστάσεις:

    • Με Παρόχους Υπηρεσιών: Μπορούμε να μοιραστούμε τα προσωπικά σας δεδομένα με παρόχους υπηρεσιών για την παρακολούθηση και ανάλυση της χρήσης της υπηρεσίας μας, για να επικοινωνήσουμε μαζί σας.
    • Για Επιχειρηματικές Μεταβιβάσεις: Μπορούμε να μοιραστούμε ή να μεταφέρουμε τα προσωπικά σας δεδομένα σας σε συνάρτησση με, ή κατά τη διάρκεια διαπραγματεύσεων για, οποιαδήποτε συγχώνευση, πώληση επιχειρηματικών περιουσιακών στοιχείων, χρηματοδότηση ή εξαγορά ολόκληρης ή μέρους της επιχείρησής μας από άλλη εταιρεία.
    • Με Συνδεδεμένες Εταιρείες: Μπορούμε να μοιραστούμε τις πληροφορίες σας με τις συνδεδεμένες εταιρείες μας, στις οποίες θα ζητήσουμε από αυτές να τηρούν αυτήν την πολιτική απορρήτου. Οι συνδεδεμένες εταιρείες περιλαμβάνουν τη μητρική εταιρεία μας και οποιεσδήποτε άλλες θυγατρικές εταιρείες, εταιρικούς συνεργάτες κοινού εγχειρίδιου ή άλλες εταιρείες που ελέγχονται από εμάς ή που ελέγχουμε εμείς.
    • Με Επιχειρηματικούς Συνεργάτες: Μπορούμε να μοιραστούμε τις πληροφορίες σας με τους επιχειρηματικούς συνεργάτες μας για να σας προσφέρουμε συγκεκριμένα προϊόντα, υπηρεσίες ή προσφορές.
    • Με άλλους χρήστες: όταν μοιράζεστε προσωπικές πληροφορίες ή αλληλεπιδράτε σε δημόσιους χώρους με άλλους χρήστες, τέτοιες πληροφορίες μπορεί να παρακολουθούνται από όλους τους χρήστες και να διανέμονται δημοσίως.
    • Με τη συγκατάθεσή σας: Μπορούμε να αποκαλύψουμε τις προσωπικές σας πληροφορίες για άλλους σκοπούς με τη συγκατάθεσή σας.

    Συνδέσεις προς άλλες ιστοσελίδες

    Η υπηρεσία μας μπορεί να περιέχει συνδέσμους προς άλλες ιστοσελίδες που δεν διαχειριζόμαστε εμείς. Εάν κάνετε κλικ σε έναν σύνδεσμο τρίτου, θα ανακατευθυνθείτε στην ιστοσελίδα του τρίτου. Σας συνιστούμε ένθερμα να εξετάσετε την πολιτική απορρήτου κάθε ιστότοπου που επισκέπτεστε.

    Δεν έχουμε κανέναν έλεγχο και δεν φέρουμε καμία ευθύνη για το περιεχόμενο, τις πολιτικές απορρήτου ή τις πρακτικές αυτών των τρίτων ιστοσελίδων ή υπηρεσιών.

    Αλλαγές σε αυτήν την Πολιτική Απορρήτου

    Μπορούμε να ενημερώσουμμε την πολιτική απορρήτου μας από καιρό σε καιρό. Θα σας ειδοποιήσουμε για οποιεσδήποτε αλλαγές δημοσιεύοντας τη νέα πολιτική απορρήτου σε αυτήν τη σελίδα. Θα σας ειδοποιήσουμε μέσω email ή/και μέσω εμφανούς ανακοίνωσης στην υπηρεσία μας πριν η αλλαγή τεθεί σε ισχύ και θα ενημερώσουμε την "Τελευταία ενημέρωση" στην κορυφή αυτής της πολιτικής απορρήτου. Συνιστάται να ελέγχετε αυτήν την πολιτική απορρήτου τακτικά για οποιεσδήποτε αλλαγές. Οι αλλαγές σε αυτήν την πολιτική απορρήτου είναι αποτελεσματικές όταν αναρτώνται σε αυτήν τη σελίδα.
